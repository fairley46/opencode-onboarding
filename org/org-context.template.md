## Approved Use

OpenCode is approved for the following types of work:

- Drafting, summarizing, and reviewing documents
- Researching and comparing options (with human review before acting)
- Answering questions about internal documentation (using approved MCP tools only)
- Generating first drafts of code, emails, or reports for human review
- Learning and onboarding exercises in this program

_Edit this list to match your organization's approved use policy._

---

## Disallowed Use

The following uses are not approved without explicit sign-off:

- Making final business, legal, security, or financial decisions
- Submitting AI-generated content as human-authored without disclosure
- Using unapproved external tools or data sources via MCP
- Processing personal data or confidential client data without a reviewed workflow
- Autonomous actions (e.g., sending emails, making purchases) without human review

_Edit this list to match your organization's restrictions._
