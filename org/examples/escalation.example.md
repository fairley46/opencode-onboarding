## Security Concerns

For suspected data incidents, unauthorized access, prompt injection concerns, or anything that feels like a security issue:

**Security Team**
Email: security@meridian.example.com
Slack: #security-incidents
Hotline: Available in the internal directory under "Security Incident Hotline"
Response time: Immediate for active incidents

If you are not sure whether something is a security concern, report it. False alarms are fine. Unreported incidents are not.

## When in Doubt

If you are not sure who to contact, start with the AI Program Team. They will route your question appropriately.

Do not proceed with a task you are uncertain about while waiting for a response. Pause the task and wait for guidance.
